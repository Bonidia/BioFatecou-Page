VOCÊ ESTÁ PRONTO PARA O FUTURO?

Este painel é usado durante a rodada da Carta Prática. Todos os jogadores participam ao mesmo tempo.

  1. Selecione o nível de ensino (Nível I ou Nível II).
  2. Escolha o modo de avaliação (com ou sem critérios específicos).
  3. O sistema sorteia automaticamente uma carta conforme as opções.
  4. Avalie as respostas e distribua os pontos.

Tabela de Critérios Gerais

Use estes critérios como apoio quando o modo de avaliação for sem critérios específicos.

Critério O que avaliar? Pontos
Engenharia de Prompt O prompt define claramente o objetivo? Limita o tipo de resposta e características exigidas no desafio? 0 - 4
Resposta final adequada O resultado realmente apresenta uma solução ou reflexão sobre o tema da carta? 0 - 2
Ética e respeito O prompt foi escrito de forma adequada? Evitou vieses preconceituosos ou comandos ofensivos? 0 - 2
Interação (Refinamento) Se a primeira resposta não foi perfeita, o aluno ajustou o comando para melhorar o resultado? 0 - 2

Dica: some até 10 pontos, se fizer sentido para a dinâmica da turma.
